**REST Assured API Automation Project
📌 Project Overview**

This project demonstrates API automation using Java, Rest Assured, and application/json content type.
The framework validates core HTTP methods including GET, POST, and PUT operations.

The project ensures end-to-end validation of API functionality, including data creation, retrieval, and update scenarios.

---

**🛠 Tech Stack**

Java

Rest Assured

TestNG

Maven

JSON (application/json)

Git

---

**📂 Test Scenarios Covered**

 GET – Retrieve All Data

 GET – Retrieve Individual Record

 POST – Create New Record

 GET – Validate Created Record

 PUT – Update Existing Record
